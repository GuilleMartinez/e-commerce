# Proyecto Javascript

El proyecto consta de una simulación de e-commerce en el cual el usuario puede realizar las siguientes operaciones:

1. Seleccionar cuántos items quiere comprar
2. Filtrar mediante categorías o por medio de formulario de búsqueda.
3. Añadir al carrito los items que desea comprar.

Por el momento solo se estuvo trabajando con el HTML + CSS y solamente se lograron definir algunas clases que se van a estar utilizando a lo largo del desarrollo.
